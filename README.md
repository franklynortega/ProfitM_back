# ProfitPlus Manufactura API

Sistema integrado con ProfitPlus Administrativo para el control de Manufactura

## Installation

Install [node.js](https://nodejs.org/en) an run the command to install Profit_M.

```bash
npm install foobar
```

## Usage

```node
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
